## 1. Logo 規範（來源：EnGenius Brand Guide PDF）

### Logo 基本元素
- **Logo 構成**：「EnGenius」字標 + Wi-Fi 訊號圖示（位於 "i" 上方）+ ® 註冊商標符號
- **Logo 特色**：Wi-Fi 訊號圖示取代了「i」上方的點，是品牌識別的核心元素

### Logo 顏色版本
| 版本 | 規格 | 使用場景 |
|------|------|---------|
| **黑色 Logo（主要）** | CMYK: 0, 0, 0, 100 / RGB: 0, 0, 0 / Web: `#000000` / Pantone Black C | 淺色/白色背景 |
| **白色 Logo（反白）** | CMYK: 0, 0, 0, 0 / RGB: 255, 255, 255 / Web: `#FFFFFF` | 深色背景、照片上方 |

### Clearspace（安全區域）
- Logo 四周必須保留足夠的「呼吸空間」
- 安全區域以 Logo 中字母的高度 "x" 為基準單位
- 四邊各留至少 1x 的距離

### 最小尺寸
| 媒介 | 最小尺寸 |
|------|---------|
| **數位（Digital）** | 高度 15px |
| **印刷（Print）** | 寬度 22mm |

### Logo 使用情境
| 背景 | 使用版本 |
|------|---------|
| 白色/淺色背景 | 黑色 Logo |
| 深色/黑色背景 | 白色 Logo |
| 藍色背景 | 白色 Logo |
| 照片背景 | 白色 Logo，放置於空白區域確保可辨識 |

### Logo 禁止事項（Don'ts）
1. **不可變形** — 不可改變 Logo 的比例、拉伸或壓扁
2. **不可改色** — 只能使用官方黑色或白色，不可使用其他顏色
3. **不可加外框** — 不可為 Logo 加描邊或外框效果
4. **不可旋轉** — 不可以任何角度傾斜或旋轉顯示
5. **不可嵌入標題** — 不可將 Logo 放在標題文字中當作文字使用
6. **不可更換字體** — 不可用其他字體重新製作 Logo
7. **不可替換字型** — 不可用不同的字體替代 Logo 的字體
8. **不可放置競爭元素** — 不可將其他商標或視覺元素放在 Logo 附近
9. **不可設定為口號** — 不可在 Logo 旁邊放置可被誤認為企業口號的文字
10. **不可突出他牌** — 不可讓其他品牌的 Logo 比 EnGenius Logo 更顯眼

### 商標聲明
> The EnGenius word and logo are trademarks of EnGenius Limited (or its subsidiaries) in the US and/or elsewhere. All rights reserved.

## 3. 色彩系統

### 品牌官方色（已確認）
| 角色 | 色碼 | 用途 |
|------|------|------|
| **品牌黑** | `#000000` (Pantone Black C) | Logo、主要文字 |
| **品牌白** | `#FFFFFF` | Logo 反白、背景 |
| **品牌藍** | `#03A9F4` | EnGenius Cloud 品牌色、導航、連結 |
| **行動橙** | `#FFA200` | 主要 CTA 按鈕、重點標示 |

### 網站使用色（從官網觀察）
| 角色 | 色碼 | 用途 |
|------|------|------|
| 深紫 | `#291734` | 頁腳、深色背景區塊 |
| 強調紅 | `#EA3D56` | 推薦標籤、促銷標示 |
| 深藍 | `#2F5BE7` | 解決方案頁面強調色 |

### 備註
- Logo 本身只有黑/白兩色，不可上其他顏色
- `#03A9F4` 為 EnGenius Cloud 品牌藍，用於 UI 與行銷素材

## 4. 字體系統（已確認）

### 官方指定字體
| 用途 | 字體 | 備註 |
|------|------|------|
| **主要字體** | **Manrope** | 標題、內文、所有行銷素材 |
| 字母間距 | `letter-spacing: .05em` | 導航與標題使用 |

### 字體風格
- Manrope：現代幾何無襯線體（geometric sans-serif）
- 風格乾淨、易讀，適合科技品牌
- 支援多種字重（Light ~ Bold），可建立清晰的層級

---

## 5. 排版風格（從官網觀察）

### 整體結構
- **最大內容寬度**：1100px，置中排列
- **導航列**：固定頂部（fixed position）
- **響應式斷點**：768px / 996px
- **區塊間距**：充足留白，區塊間有明確分隔

### 資訊層級
| 層級 | 字體大小 | 用途 |
|------|---------|------|
| H1 主標題 | ~42px | Hero 區域標題 |
| H2 次標題 | 中號 | 區塊標題 |
| 內文 | 標準 | 描述文字 |

### 排版模式
- **Hero 區域**：全寬背景圖 + 深色半透明遮罩 + 置中大標題 + CTA 按鈕
- **內容區塊**：交替使用「左圖右文」與「右圖左文」
- **產品展示**：網格卡片排列（桌面 2-4 欄，手機 1 欄）
- **功能展示**：3-5 個並排卡片 + icon

## 8. 整體設計調性

**科技專業 × 簡約現代**

- 強調「AI-Driven Cloud」的技術先進性
- 配色以藍 + 橙為主軸，科技感強
- 排版乾淨、留白充足，適合 B2B 閱讀
- 深色區塊營造高級感
- CTA 按鈕用橙色跳出，引導行動
