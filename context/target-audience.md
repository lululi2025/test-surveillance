# EnGenius Target Audience — 目標客群輪廓

> 版本：v1.0（2026-03-20）
> 來源：官網產品頁面、解決方案頁面、Cloud 平台頁面
> 狀態：初版，待業務 / 行銷團隊補充實際客戶 feedback

---

## 概述

EnGenius 的產品橫跨 B2B 與 B2C，但核心營收與行銷資源集中於 **B2B 市場**。主要受眾可分為四大 Persona，再依垂直產業細分應用場景。

---

## Persona 1：IT 管理者（SMB / Mid-Market）

> 「我需要企業級的網路品質，但沒有企業級的預算和人力。」

### 基本輪廓

| 項目 | 描述 |
|------|------|
| 角色 | IT Manager / Network Administrator / IT Director |
| 公司規模 | 10–500 人（中小企業至中型企業） |
| 技術背景 | 中高，具備基礎網路知識，但不一定有專職網路工程師 |
| 決策權限 | 技術選型主導者，向上呈報預算審批 |
| 日常工具 | Cloud dashboard、NMS、RMM 平台 |

### 痛點（Pain Points）

- 管理多品牌設備耗時，缺乏統一介面
- 傳統企業方案（Meraki、Aruba）授權費用高，年年續約壓力大
- 新據點部署需要到場，人力與差旅成本高
- 網路問題排查困難，缺乏即時可視化工具
- 一人 IT 或小團隊，無法處理複雜架構

### 購買動機

- 降低網路管理的時間與人力成本
- 用有限預算達到企業級可靠度
- 支援遠端管理，不需每次到場
- 簡化多據點統一管理

### EnGenius 對應訊息

| 訊息主軸 | 文案方向 |
|----------|----------|
| 統一管理 | "One cloud platform to manage all your network devices — APs, switches, and gateways." |
| 成本可控 | "Enterprise-grade networking without mandatory licensing fees." |
| 零接觸部署 | "Deploy new sites remotely with zero-touch provisioning — no truck rolls needed." |
| 即時可視化 | "Real-time monitoring, troubleshooting, and analytics from anywhere." |

### 內容偏好

- 產品比較表（vs Meraki / Aruba / Ubiquiti）
- TCO 計算工具或案例
- 技術白皮書、部署指南
- 線上 Demo / Webinar

## Persona 2：SI / MSP 系統整合商

> 「我要能幫客戶快速部署、集中管理，而且有足夠的利潤空間。」

### 基本輪廓

| 項目 | 描述 |
|------|------|
| 角色 | MSP Owner / Solutions Architect / Technical Sales |
| 公司規模 | 5–100 人的服務型公司 |
| 管理客戶數 | 10–500+ 個客戶站點 |
| 技術背景 | 高，熟悉多廠牌網路設備與雲端管理平台 |
| 決策考量 | 利潤率、管理效率、客戶滿意度、品牌支援力度 |

### 痛點（Pain Points）

- 客戶數量多，逐一管理效率低
- 每次部署都需到場，差旅成本侵蝕利潤
- 競爭激烈，純硬體轉售利潤薄
- 現有平台 API 封閉，無法與自家系統整合
- 授權費用轉嫁客戶困難，影響報價競爭力

### 購買動機

- 多租戶管理平台，一個 dashboard 看所有客戶
- API 開放整合，嵌入既有 RMM / PSA 工具
- 無強制授權費 = 更高利潤或更有競爭力的報價
- 零接觸佈建省下現場部署成本
- 可靠的 Partner Program 支援

### EnGenius 對應訊息

| 訊息主軸 | 文案方向 |
|----------|----------|
| 多租戶管理 | "Manage hundreds of client sites from a single cloud dashboard." |
| API 整合 | "Open API architecture — integrate with your existing RMM, PSA, or custom tools." |
| 利潤空間 | "No mandatory licensing means better margins for you and competitive pricing for your clients." |
| Partner 支援 | "Elevate Partner Program — enabling, empowering, and elevating our partner community." |
| 白牌潛力 | "Customize themes, frontend, and modules to make the platform your own." |

### 內容偏好

- Partner Program 介紹與成功案例
- API 文件與整合範例
- 利潤模型試算
- Co-marketing 素材與 MDF 說明
- 技術認證課程

## Persona 3：企業決策者（C-suite / Business Owner）

> 「這個投資能幫公司省多少錢？多快能回本？」

### 基本輪廓

| 項目 | 描述 |
|------|------|
| 角色 | CEO / CFO / COO / VP of Operations |
| 公司規模 | 50–1000+ 人 |
| 技術背景 | 低至中，關注商業成果而非技術規格 |
| 決策權限 | 最終預算核准者 |
| 關注指標 | ROI、TCO、營運效率、風險控管 |

### 痛點（Pain Points）

- IT 基礎架構支出持續上升，不清楚是否合理
- 數位轉型壓力，但不想過度投資
- 多據點營運效率低，管理成本高
- 資安風險增加，但資安預算有限

### 購買動機

- 明確的成本節省與 ROI
- 簡化營運、減少 IT 人力需求
- 支援業務擴展（開新店、新據點）不需等比增加 IT 成本
- 整合安控（AI Camera）降低另購監控系統的支出

### EnGenius 對應訊息

| 訊息主軸 | 文案方向 |
|----------|----------|
| TCO 優勢 | "Cut networking costs by eliminating mandatory annual licensing — without sacrificing performance." |
| 營運效率 | "Enable your IT team to manage all locations remotely — reduce truck rolls and downtime." |
| 擴展性 | "Scale your network as your business grows — add sites in minutes, not weeks." |
| 整合安控 | "Unified networking and AI-powered surveillance under one cloud platform." |

### 內容偏好

- ROI / TCO 案例分析（一頁式摘要）
- 客戶成功故事（同產業）
- 高層次 Solution Overview（非技術細節）
- 簡報用投影片

## Persona 4：通路夥伴 / 經銷商（Reseller / VAR / Distributor）

> 「哪個品牌能讓我好賣、好裝、好服務？」

### 基本輪廓

| 項目 | 描述 |
|------|------|
| 角色 | Sales Manager / Product Manager / Procurement |
| 公司類型 | 區域經銷商、VAR、分銷商 |
| 技術背景 | 中，了解產品規格但不深入架構 |
| 決策考量 | 產品競爭力、利潤率、售後問題少、品牌行銷支援 |

### 痛點（Pain Points）

- 高階品牌利潤薄，低階品牌售後問題多
- 客戶常拿 Ubiquiti 價格來比，但要求企業級功能
- 缺乏原廠行銷素材與銷售工具
- 產品教育訓練不足，銷售信心低

### 購買動機

- 中間價位帶 + 企業功能 = 好推好賣
- 原廠提供行銷素材與 co-marketing 資源
- 產品穩定、退貨率低
- 技術支援回應快

### EnGenius 對應訊息

| 訊息主軸 | 文案方向 |
|----------|----------|
| 價格定位 | "Enterprise features at a price point that wins deals — sweet spot between premium and budget." |
| 銷售支援 | "Ready-to-use marketing materials, sales tools, and co-marketing funds." |
| 產品穩定 | "Reliable hardware backed by responsive technical support." |
| 差異化賣點 | "No licensing fees — the competitive edge your customers are looking for." |

### 內容偏好

- 產品型錄與比較簡報
- 經銷價格表與利潤計算
- Quick Start Guide（給客戶的安裝簡易指南）
- 銷售話術卡 / Battle Card

## Persona 5：Residential / SOHO 用戶（次要客群）

> 「我只想要穩定的 Wi-Fi，設定越簡單越好。」

### 基本輪廓

| 項目 | 描述 |
|------|------|
| 角色 | 家庭用戶 / 自由工作者 / 微型企業主 |
| 技術背景 | 低，不想碰進階設定 |
| 預算敏感度 | 高，與消費級品牌比較 |
| 決策方式 | 線上評價、Amazon 比價、朋友推薦 |

### 痛點

- Wi-Fi 死角、訊號不穩
- 消費級路由器功能不足但企業級太複雜
- 不想付月費或訂閱

### EnGenius 對應訊息

| 訊息主軸 | 文案方向 |
|----------|----------|
| 簡單設定 | "Business-grade Wi-Fi you can set up in minutes — no IT degree required." |
| 無月費 | "Simple, secure Wi-Fi — no subscriptions, no hidden fees." |
| App 管理 | "Manage your home network from the FitXpress app, anytime, anywhere." |

### 產品對應

- EnGenius Fit 系列（FitXpress app-based 管理）

## 垂直市場 × Persona 對應矩陣

下表呈現各垂直市場中，最常接觸的決策角色與關鍵訴求：

| 垂直市場 | 主要 Persona | 關鍵訴求 | 產品重點 |
|----------|-------------|----------|----------|
| Hospitality（飯店） | IT 管理者 / 決策者 | 賓客體驗、全館覆蓋、集中管理 | Cloud AP + Switch + Gateway |
| Education（教育） | IT 管理者 | E-Rate 合規、校園安全、預算控制 | Cloud AP + Switch |
| Healthcare（醫療） | IT 管理者 / 決策者 | 可靠連線、通訊整合、合規需求 | Cloud AP + Voice + Camera |
| MDU（集合住宅） | MSP / SI | 多租戶管理、批量部署、低維運 | Cloud AP + Switch + Fit |
| Campgrounds / RV | MSP / SI | 戶外大範圍覆蓋、遠端管理 | Outdoor AP + PtP |
| Retail（零售） | 決策者 / IT 管理者 | 顧客 Wi-Fi、AI 監控防損、多店管理 | Cloud AP + AI Camera |
| Warehouse（倉儲） | IT 管理者 | 大空間覆蓋、高密度連線、環境耐受 | Industrial AP + Switch |

## 行銷素材產出指引

根據以上 Persona，建議不同素材的語氣與深度：

| 素材類型 | 主要對象 | 語氣 | 技術深度 |
|----------|---------|------|----------|
| Landing Page | IT 管理者 + 決策者 | 專業、簡潔 | 中（功能導向） |
| Technical White Paper | IT 管理者 + MSP | 技術嚴謹 | 高（規格 + 架構） |
| Case Study | 決策者 + IT 管理者 | 故事性、數據佐證 | 中 |
| EDM / Email Campaign | 全客群 | 簡短、CTA 明確 | 低～中 |
| Partner Program Page | MSP / 經銷商 | 商業導向、利潤強調 | 低（商業價值） |
| Social Media Post | 全客群 | 簡潔、視覺為主 | 低 |
| Sales Battle Card | 經銷商 | 比較式、重點條列 | 中 |
| Brochure / Catalog | 經銷商 + IT 管理者 | 圖文並茂、規格表 | 中～高 |
| Exhibition Backwall | 決策者 + IT 管理者 | 視覺衝擊、slogan 為主 | 低（印象導向） |
